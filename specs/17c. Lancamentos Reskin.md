# Spec 17c — Lançamentos Re-skin

## Status: 🚧 In Progress

## Overview

Fourth spec of **Fase A**. Rebuilds `/lancamentos` to match the prototype:
PageHeader · QuickAddBar (visual) · filter Card with type chips · table
grouped by month with class-colored bar + sign-colored net amount · row
click opens a slide-over detail panel.

**No backend changes.** Uses the existing `Lancamento` entity, `LancamentoType`
enum (9 values), and `LancamentoModal` for edit/create flows.

The Lançamentos table currently mixes 6 position-affecting types (COMPRA,
VENDA, BONIFICAÇÃO, SUBSCRIÇÃO, COME_COTAS, RESGATE_TOTAL) with 3 income
types (DIVIDENDO, JUROS, JCP) that will split out into Distribution at
spec 08. The chips show all 9 today, with a one-line note in the spec doc
that the 3 income chips will migrate to the future `/proventos` page.

---

## In Scope

### Visual changes

- **PageHeader** with `Lançamentos`, count + label `movimentações · X ativos`, action button slot for `+ Novo Lançamento`
- **QuickAddBar** placeholder input (single-line, visual only — opens the existing modal when clicked; future spec parses natural language)
- **Filter Card** matching the prototype:
  - Top row: SearchInput · date-range (two existing inputs styled tighter) · ToggleSwitch "Incluir inativos"
  - Separator
  - `FilterGroup` "Tipo" with `MultiChips` for all 9 LancamentoType codes, colored per type
  - Separator
  - Stats footer: `X lançamentos · Net total · ±R$ Y,Z` (sign-colored)
- **Table grouped by month** (YYYY-MM headers like "Maio 2026"), with:
  - Data (tnum, gray)
  - Tipo (small colored badge per type)
  - Ativo: vertical class-color strip + ticker (mono) + name (gray below)
  - Qtd (tnum right)
  - Preço unit. (tnum money muted right)
  - Net (tnum money with sign, green/red/gray based on sign)
  - Moeda (CcyPill, replaces FX column for now)
  - Row click → slide-over

### Slide-over `LancamentoDetailPanel`

Mirrors `AssetDetailPanel`'s shell:
- Sticky header: class bar · type badge · ticker/name · CcyPill · close
- Asset block: linked card showing asset name + ticker + class + custódia
- Fields grid: Data, Settlement date (if any), Quantidade, Preço unit., Bruto, Taxa, Imposto, Líquido, Moeda, FX rate, Nota de negociação
- Notes (textarea readonly, falls back to "—" if null) — Attachment list is **out of scope** (Attachment table comes with spec 15)
- Footer: Editar (opens existing LancamentoModal) · Desativar (existing confirm dialog)

### Primitives added to `components/ui.tsx`

- `QuickAddBar` — single-line bar with leading sparkles icon + placeholder text + Enter hint; renders a click target that triggers a callback
- `TypeBadge` — colored uppercase badge for a Lançamento type code
