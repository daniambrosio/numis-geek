## Overview

Substitui o card antigo de Proventos no Dashboard por um card mais
informativo: chart compact em cima + lista "Por tipo" com os 5 tipos
sempre visíveis (mesmo zerados) embaixo.

## In Scope

### 1. Novo card

`frontend/src/pages/Dashboard.tsx` — substituir a célula Proventos:

```tsx
<Card span={4} title="Proventos">
  <ProventosChart
    defaultBreakdown="klass"
    defaultCurrency="BRL"
    defaultPeriod="12m"
    includeSynthetic={syntheticOnDashboard}
    onIncludeSyntheticChange={setSyntheticOnDashboard}
    compact
    hideToggles
    noCard
  />
  <ProventosTypeList
    includeSynthetic={syntheticOnDashboard}
    currency="BRL"
    period="12m"
  />
</Card>
```

### 2. Componente `<ProventosTypeList />`

`frontend/src/components/ProventosTypeList.tsx` — versão compacta do
`<ProventosByTypeCard />` (Spec 32):

```
Por tipo
● Dividendo                75%  R$ 32 mil
● Juros                    12%  R$ 5 mil
● JCP                       8%  R$ 3 mil
● Aluguel                   3%  R$ 1 mil
● Prêmio sintético [OPÇÕES] 2%  R$ 1 mil
```

- Sempre 5 linhas, ordem fixa (DIVIDEND → INTEREST → JCP →
  SECURITIES_LENDING → OPTION_PREMIUM).
- Zero é OK (renderiza `R$ 0` + `0%`).
- Quando `includeSynthetic=false`, linha OPTION_PREMIUM fica dim
  (text-slate-400) + texto "desligado" no lugar do %.

Sem o card wrapper — assume que está dentro de um `<Card>` parent.

### 3. Span / size

- O card Proventos no Dashboard ocupa `span=4` (4 de 12).
- Chart compact: max-height ~140px.
- Lista: max-height ~180px.
- Total card: ~360px de altura → cabe ao lado de outros cards na grid.

### 4. State local ao Dashboard

`syntheticOnDashboard` é estado local da página Dashboard, default
`true`. Não persiste (cada visita à página resseta).

Decisão: **não compartilhar** state entre Dashboard e Distributions —
são visualizações independentes; cada uma com seu toggle.

### 5. Click-through

Click em qualquer linha de `<ProventosTypeList />` → navega pra
`/distributions?type={KEY}` (filtro pré-selecionado na lista).

Click no chart compact (em qualquer barra) → navega pra
`/distributions?ym={YYYY-MM}` (V1.5, opcional).

## Out of Scope

- Persistência do toggle `syntheticOnDashboard` por user.
- Drill-down completo (filtro por barra) — V1.5.
- Mudança de outros cards do Dashboard (Portfolio, KPIs).

## Acceptance Criteria

1. Card Proventos no Dashboard mostra chart compacto sem toggles e
   sem legenda.
2. Abaixo do chart, lista de 5 tipos sempre visível.
3. Toggle "Incluir sintéticos" não aparece no Dashboard (porque
   `hideToggles` no chart e lista usa o state local).
4. Quando `syntheticOnDashboard=false`: linha OPTION_PREMIUM dim
   ("desligado").
5. Click em linha vai pra `/distributions?type=...`.
6. Build limpo.
