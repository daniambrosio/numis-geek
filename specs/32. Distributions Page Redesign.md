### 2. Componente `<ProventosByTypeCard />`

`frontend/src/components/ProventosByTypeCard.tsx`:

```typescript
interface Props {
  includeSynthetic: boolean;
  currency: Currency;        // BRL | USD
  period: Period;            // pra consistência com o chart
}
```

Internamente consome `api.getDistributionsChart({ breakdown: "type", currency, period, include_synthetic: true })`
sempre (precisa do total inclusive sintético pra calcular %).

Renderização:
- Grid responsivo: 2 cols (mobile) → 3 (md) → 5 (lg).
- Ordem fixa: DIVIDEND → INTEREST → JCP → SECURITIES_LENDING → OPTION_PREMIUM.
- Cada chip:
  ```
  ● {label}  [OPÇÕES if synthetic]
  {fmtChart(total, ccy)}
  {pct}%
  ```
- Se `key === "OPTION_PREMIUM"` e `includeSynthetic=false`:
  - `border-dashed`
  - `bg-slate-50` (mais claro)
  - `opacity-60`
  - sub-text "desligado" em vez do %
- Badge `[OPÇÕES]` é Tailwind pill purple, só no chip sintético.
