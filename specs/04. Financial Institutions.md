## Overview

Financial institutions (IFs) are system-level entities shared across all workspaces вЂ” not owned by any workspace. A new `sysadmin` role allows a super-user to manage system-level data and has visibility into all workspace data.

## Role: sysadmin

- Added to `UserRole` enum alongside `admin` and `member`
- Sysadmin users have `workspace_id = None`
- Created via `scripts/seed.py` with email `sysadmin@numis-geek.internal`
- Password from `.env` `SYSADMIN_PASSWORD` (default: `changeme123`)
- Sees a "Sistema" section in the sidebar with system-level management pages

## API

| Method | Path | Auth | Description |
|---|---|---|---|
| GET | `/financial-institutions` | any authenticated | List active IFs |
| POST | `/financial-institutions` | sysadmin | Create IF |
| PUT | `/financial-institutions/{id}` | sysadmin | Update IF |
| PUT | `/financial-institutions/{id}/deactivate` | sysadmin | Soft-delete |

Audit actions: `financial_institution.created`, `financial_institution.updated`, `financial_institution.deactivated`
