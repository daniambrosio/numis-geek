# Spec 09 — Asset cleanup: class collapse 14→11 + country + current_price

## Status: 🚧 In Progress

## Overview

Second spec of **Fase B**. Three changes on `Asset`:

1. **Collapse asset_class 14 → 11.** Merge STOCK_BR + STOCK_US → STOCK;
   FII + REIT → REIT; BOND + FIXED_INCOME → FIXED_INCOME. The BR vs US
   distinction moves out of the class into a dedicated `country` field.

2. **Add `country` (ISO-2).** Required field, NOT NULL. Backfilled from the
   old asset_class + (for ETF/CASH) the asset's currency.

3. **Add `current_price` (Numeric) + `price_updated_at` (DateTime).**
   Both nullable until the user starts setting them. Unblocks all the
   `—` placeholders on Ativos table, Asset detail panel, and the
   Dashboard's Top movers card.

---

## Backfill rules

| Old class | New class | New country | Source |
|---|---|---|---|
| `STOCK_BR` | `STOCK` | `BR` | direct |
| `STOCK_US` | `STOCK` | `US` | direct |
| `FII` | `REIT` | `BR` | direct |
| `REIT` | `REIT` | `US` | direct |
| `BOND` | `FIXED_INCOME` | `US` | Avenue holds US Treasuries / corporate bonds |
| `FIXED_INCOME` | `FIXED_INCOME` | `BR` | CDB / Tesouro / LCI |
| `ETF` | `ETF` | `currency == USD ? US : BR` | BR ETFs (BOVA11) keep BR; US ETFs (SPY) → US |
| `FUND` | `FUND` | `BR` | All BR funds with CNPJ |
| `CRYPTO` | `CRYPTO` | `currency == USD ? US : BR` | Coinbase USD → US; Mercado Pago BRL → BR |
| `REAL_ESTATE` | `REAL_ESTATE` | `BR` | user's properties |
| `VEHICLE` | `VEHICLE` | `BR` | user's vehicles |
| `CASH` | `CASH` | `currency == USD ? US : BR` | Avenue float USD → US; others → BR |
| `FGTS` | `FGTS` | `BR` | always |
| `PRIVATE_PENSION` | `PRIVATE_PENSION` | `BR` | always BR pension |

Expected output for the local DB (181 assets):
- STOCK: 54 (30 BR + 24 US)
- REIT: 33 (28 FII BR + 5 REIT US)
- FIXED_INCOME: 42 (27 BR + 15 BOND US)
- ETF: 23 (5 BR + 18 US)
- FUND: 7 BR · PRIVATE_PENSION: 7 BR · CRYPTO: 3 (2 US + 1 BR)
- CASH: 5 (4 BR + 1 US) · REAL_ESTATE: 2 BR · VEHICLE: 2 BR · FGTS: 3 BR
