## Overview

Adiciona coluna **"Atualizado"** na tabela de Ativos da página `/assets`,
antes da coluna de ações. Cada linha mostra dot colorido + idade
relativa + botão de refresh inline. Permite consertar uma cotação
travada sem sair da tabela.

## In Scope

### 1. Componente: célula `<PriceCell />`

```tsx
function PriceCell({ asset }: { asset: AssetOut }) {
  const tier = asset.price_tier;
  const age = asset.price_updated_at
    ? formatRelative(asset.price_updated_at)
    : "—";
  const sourceLabel = SOURCE_LABEL[asset.price_source ?? "MANUAL"];

  return (
    <div className="flex items-center gap-2">
      <Tooltip content={`${sourceLabel} · ${asset.price_updated_at ?? "nunca"}`}>
        <Dot color={TIER_COLOR[tier]} />
      </Tooltip>
      <span className="text-xs text-slate-500">{age}</span>
      {asset.price_source !== "MANUAL" && (
        <IconButton
          icon="refresh"
          onClick={(e) => {
            e.stopPropagation();
            refreshOne(asset.id);
          }}
          loading={refreshing[asset.id]}
        />
      )}
    </div>
  );
}
```

### 2. Hook `useRefreshOne()`

Endpoint: `POST /api/assets/{id}/refresh-price`. Atualiza linha
otimisticamente; toast em erro com `reason`.

### 3. Coluna na tabela

`frontend/src/pages/Assets.tsx` (ou `AssetsTable.tsx`) — adicionar
header "Atualizado" entre o penúltimo grupo de colunas e a coluna de
ações. Width fixa (~140px); alinhamento esquerda.

### 4. Estado de loading por linha

Tracking por asset_id em estado local:

```typescript
const [refreshing, setRefreshing] = useState<Record<string, boolean>>({});
```

Após sucesso, recarregar só a linha (mutate cache do `useAssets`) —
evita refetch da lista inteira.

### 5. Acessibilidade

- Tooltip tem `aria-label` com timestamp ISO formatado em pt-BR
- Botão refresh tem `aria-label="Atualizar preço de {ticker}"`
- `e.stopPropagation()` no botão impede que click na linha abra
  AssetDetail por acidente

### 6. Sort (V1.5, escopo seguinte)

Pré-arquitetar a coluna para suportar sort por `price_updated_at ASC`
no futuro — mas não habilitar UI agora.

## Out of Scope

- Sort UI por staleness (V1.5)
- Bulk action "Atualizar selecionados" (V1.5)
- Inline edit de preço manual aqui — fica no AssetDetail (spec 28)

## Acceptance Criteria

1. Coluna "Atualizado" aparece na tabela com dot + idade + ícone
   refresh (este só pra automatizados).
2. Click no ícone refresh dispara endpoint, mostra spinner,
   atualiza dot+idade ao final, sem reload da lista inteira.
3. Click no ícone NÃO abre AssetDetail.
4. Tooltip mostra `{Source label} · {ISO timestamp}`.
5. Asset com `price_source = MANUAL` mostra dot+idade mas **sem** o
   botão refresh.
6. Asset com `price_updated_at = null` mostra dot cinza
   (UNKNOWN) + "—".
7. Build limpo.
